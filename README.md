# E-commerce Platform — ES/EN

## Español

E-commerce backend con catálogo y órdenes:

- Productos (SKU, precio)
- Órdenes + items + total
- IA opcional para “asistente de compra” vía `/api/v1/ai/chat`

### Endpoints
- `/api/v1/shop/products` (GET/POST)
- `/api/v1/shop/orders` (GET/POST, list)

## English

E-commerce API with products and orders. Optional AI shopping assistant.
